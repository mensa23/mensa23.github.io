什么是 Project Orion

Project Orion 是冷战早期，美苏刚开始太空竞赛的时候，美国科学家发起的。他们计划建造一艘像远洋巨轮般庞大的宇宙飞船——“猎户座”，使用数千枚小型核弹作为动力，将这艘至少可以乘坐40名航天员的飞船送到地球临近的行星。根据计划，“猎户座”飞船只需要125天就可以往返地球和火星，预计1970年搭载宇航员登陆土星。

Project Orion 几乎没有任何明显的技术缺陷，然而，它却有一个最大的弱点，那就是它依赖于原子弹爆炸做动力，当它飞出大气层时，必将释放出核辐射尘污染地球环境。也因为这个弱点，该计划最终流产。

关于这个博客

这个网站和 Project Orion 没有任何关系，博主只是个喜欢科幻的程序员，并且认为猎户座计划是一个很夸张但是又很牛逼的想法。我希望自己有一个牛逼的不被传统观念束缚的人生，以及实现一些夸张的想法。而你可以在这里看到一些没有逻辑性的观点和文章，这些随笔有一部分会得到你的认可，也有一部分会让你觉得难受，但无论你有什么体会，我都希望能够给你带来一点收获。宇宙起源于大爆炸，思维也是如此，愿你的人生与众不同。

关于坑爹的作者

完全没有任何值得一提的地方，十足的逗逼，写出来的东西自己都不敢看，写作的巅峰期是高考前完全按照套路来的八股文，初中阶段曾一度被眼瞎的语文老师评价为当代梁启超，可见我国教育事业搞得有多烂。幼儿园时期的国画水平超越授课的老师，作品被老师篡改名字发表到了报纸上，从此对艺术失去信心，走上计算机这条不归路。目标是成为与众不同的程序员，差不多属于文艺型技术宅这种。
